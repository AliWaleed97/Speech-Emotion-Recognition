Details

Aim

The aim of this project is to add one or more metrics to a current emotions speech recognition system. The system current metrics are Laughing, Anger, Irritation.Arousal and Gender. One new metric could be Age

Tasks

  1. Conducting the literature review on emotions in speech recognition platforms, and exploring methodologies of building them.
  2. Investigating existing and related work on emotions in speech recognition platforms.
  3. Understanding the current emotions in speech recognition platform provided by Affectiva.
  4. Designing the classifier to the provided data set.
  5. Implementing and training feature extraction mechanism for the new metrics.
  6. Platform integration and testing.
  7. Performance analysis and evaluation.

Process

  1. Speech signal acquisition
  2. Features extraction
  3. Feed the features into a classifier and predict an emotion
